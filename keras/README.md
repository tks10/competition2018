# competition2018 Keras
これはKerasを利用したテンプレートです．

## 環境
TensorFlowについては，CPU版またはGPU版のどちらかをインストールしてください．
KerasはTensorFlow内蔵のものを利用しますが，パッケージを入れても大丈夫です．

| パッケージ名 | バージョン |
|:-----------:|:---------:|
| Python | 3.5 |
| tensorflow | 1.7 |
| tensorflow-gpu | 1.7 |
| keras | 2.1.5 |

## 実行方法
`python template_keras.py [画像を保存したディレクトリ]`  
**画像のディレクトリ構造は以下のようにする**  
images(ここの名前は任意)  
└ target  
└ others
